# Zara OS Product Roadmap

This document outlines the development trajectory of Zara OS from a single-practice tool at Zara Medical to a multi-tenant enterprise system.

## Phase 1: Foundation & Local Validation (Q2 2026)
- [x] Initial monorepo scaffolding.
- [x] Basic agent routing logic and EHR lookups.
- [x] Automated compliance gates in CI/CD pipeline.
- [ ] Integration with local sandbox EHR.

## Phase 2: Pilot Practice Onboarding (Q3 2026)
- [ ] Multi-tenant organization partitioning.
- [ ] Integration with Twilio Voice/SMS active pipelines.
- [ ] Onboard first 3 partner practices on pilot billing models.

## Phase 3: Prior-Auth & Referral Expansion (Q4 2026)
- [ ] Launch `apps/practice-ops` module.
- [ ] Prior authorization automation agent (integrating with Change Healthcare APIs).
- [ ] Automated patient referral tracking and calendar invites.

## Phase 4: Scale and Enterprise Hardening (Q1 2027)
- [ ] HITRUST CSF Certification prep.
- [ ] Full integration with Epic App Market and Athenahealth API platforms.
